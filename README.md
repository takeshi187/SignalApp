# SignalApp
## Пользовательская документация SignalApp
1. Описание приложения
SignalApp — WPF‑приложение для генерации и анализа сигналов (Sine, Square, Triangle, Sawtooth). Позволяет строить график сигнала, выполнять расчет статистических параметров, сохранять данные в файл и базу данных.
2. Системные требования
* Операционная система: Windows 10 или Windows 11
* Версия .NET: .NET 8 или выше
* Оборудование: не менее 4 ГБ ОЗУ, 100 МБ свободного места.
3. Инструкция по установке и запуску 
* Склонировать или скачать репозиторий;
* Следовать иснтрукции user_manual.docx
